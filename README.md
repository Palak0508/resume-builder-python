# Resume Builder using Python

## About the Resume Builder

The Resume Builder is a project that I made using Python. This project lets people enter their details academic details and skill details through the terminal. Then it automatically generates a resume in a text file.

My main goal for this project was to make it easy for beginners to create a resume. I also wanted to show how basic programming concepts can be used in a real-world application. The Resume Builder is a tool for people who are new to programming and want to build something useful.

---

## The Problem

A lot of students have trouble making a structured resume. The Resume Builder solves this problem by giving them a tool that generates a resume automatically. This way students can focus on things and not worry about making a resume from scratch.

---

## Features of the Resume Builder

* The Resume Builder has a user- terminal interface that is easy to use

* It takes input for sections so you can add all your details

* The Resume Builder generates an formatted resume that looks great

* It saves your resume as a file so you can easily open it and print it

* The Resume Builder uses input validation with if- statements to make sure everything is correct

* After creating your resume it shows you where the file is saved so you can find it easily

---

## Technologies Used in the Resume Builder

* I used Python to build the Resume Builder

* I also used some concepts like:

* Input and output so you can enter your details and see the output

* Conditional statements like if-else to make sure everything is

* File handling so the Resume Builder can save your resume as a file

---

## How to Run the Resume Builder

1. First,. Clone the repository

2. Then open the project folder

3. Next run the Python file by typing:

```

python resume_builder.py

```

4. After that enter your details when prompted

5. Finally your resume will be saved as resume.txt and you can open it and print it
